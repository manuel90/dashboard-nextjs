# Demo Dashboard Growth of COVID-19

## Quick start

- Clone the repo: `https://github.com/manuel90/dashboard-nextjs.git`.
- `npm install` (Using Node Version: 14.10.1)
- `npm run start`
- Go to `http://localhost:3000`


### Demo

- https://dashboardnextjs.mlara.dev
